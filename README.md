## Featured Projects

### 🤖 [AI-Powered Job Search Automation](https://github.com/harichardson68/job-search-hans)
A fully autonomous Python pipeline that aggregates jobs from 6 APIs daily, scores them with a weighted multi-track classification engine, generates AI cover letters via Claude API, and delivers a ranked email digest. 

The system features a **human-in-the-loop feedback loop** — decisions submitted via Google Forms trigger a midnight script that reads Google Sheets, auto-patches filter logic, updates scoring weights, and commits everything to GitHub. The system rewrites parts of its own source code based on real-world feedback.

`Python` `Claude API` `Google Sheets API` `SMTP` `Git Automation` `Self-Improving`
